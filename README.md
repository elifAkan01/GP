# 🍽️ Yemek Kartı Uygulaması

Bu proje, C# WinForms kullanılarak geliştirilmiş bir yemek kartı uygulamasıdır.

## 🚀 Özellikler

- 📌 Yeni yemek ekleme
- 🥗 Malzeme ekleme
- 🖼️ Yemek görseli ekleme
- 🧾 Yazdırma (Print Preview)
- 🗄️ SQL Server veri tabanı bağlantısı

## 🛠️ Kullanılan Teknolojiler

- C# (WinForms)
- SQL Server
- ADO.NET

## 📂 Proje Yapısı

- `YemekKarti` → Ana proje
- `NotePad1` → Önceki çalışma
